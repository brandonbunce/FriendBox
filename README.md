# FriendBox

FriendBox is an interactable device that can be used to draw, send, and view images sent to and from your friends!
More features to come...

- [x] Drawing
- [x] Drawing Menu GUI
- [x] Save Slots
- [x] Additional Tools (Dither, Fill, Pencil)
- [ ] Sticker
- [ ] Home GUI
- [ ] Settings GUI
- [ ] Sending Capabilities
- [ ] Stickers
- [ ] Registration GUI
- [ ] Memories
- [ ] Send History
- [ ] Rotate Gestures
- [ ] OTA Updates
